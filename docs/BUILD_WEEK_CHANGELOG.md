# Build Week change boundary

## Before July 13, 2026

The private Omyear project already had a warm editorial visual system, an Astro
template for handcrafted personal books, one-off private datasets, person-specific
calculation experiments, and web/print presentation work. Those private materials are
not included here and are not presented as Build Week work.

## Added during the submission period

1. Generic strict input schema and synthetic English questionnaire.
2. Generalized deterministic calculation engine and immutable source registry.
3. Source-bounded GPT‑5.6 Structured Output prompt and schema.
4. Responses API adapter with a privacy-preserving safety identifier.
5. Verifier for citation membership, exact values, structure and language.
6. Mandatory human-review boundary and compiler into the Astro book format.
7. Public pipeline page, in-book provenance ledger and GPT response evidence.
8. Self-contained E2E, responsive QA and sanitized submission repository.
