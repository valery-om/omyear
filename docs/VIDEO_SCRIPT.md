## 1:03–1:28 — From facts to generation

**Screen:** Questionnaire, explicit consent, and the GPT-5.6 generation state.

> Creating a new book begins with facts, not inference. The person chooses the
> birthday that opens the year, provides birth details, selects places to question,
> and adds only the context they want the book to consider. Consent is explicit.
> Deterministic code performs the calculations. Then GPT-5.6 connects only those
> calculated results with the chosen life context, shaping them into a coherent book

**Screen:** Generated result, atlas, chapter content, and source ledger.

> This is a real end-to-end generation, not a prerecorded mockup. The finished result
