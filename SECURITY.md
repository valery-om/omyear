# Security and privacy

This public repository may contain only application code, the invented Maya demo,
sanitized aggregate evidence, generic brand assets and documentation.

Never commit API keys, `.env.local`, real questionnaires, recipient photos, music,
PDFs, deployment credentials or `pipeline/runs/` from a real person.

If private data or a credential is discovered, stop distribution, rotate the affected
credential, remove the material from the public repository and its history, and
rebuild from the synthetic sample.

Generated drafts require human editorial review. The product must not present
symbolic systems as scientific evidence, guaranteed outcomes, professional advice or
instructions for high-impact decisions.
